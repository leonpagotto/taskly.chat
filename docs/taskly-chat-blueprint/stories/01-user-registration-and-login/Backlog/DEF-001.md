---
id: DEF-001
title: Define Authentication Policies and Requirements
responsibleArea: Product Manager
---
Define and refine the product-level requirements for user authentication, security, and privacy. This task includes:
*   Specifying password strength requirements (e.g., minimum length, character types).
*   Defining password reset policies (e.g., token expiry, single-use tokens).
*   Outlining session management policies (e.g., JWT expiry, refresh token strategy).
*   Ensuring compliance with GDPR, CCPA, and other relevant data privacy regulations.
*   Reviewing and approving the UI/UX designs and technical implementation plans to ensure alignment with product vision and security standards.
