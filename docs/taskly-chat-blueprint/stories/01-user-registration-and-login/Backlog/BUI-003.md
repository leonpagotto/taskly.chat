id: BUI-003
title: Build Password Reset UI (Web & Desktop)
responsibleArea: Frontend Developer (Web & Desktop)
---
Develop the user interface for the forgotten password and password reset process. This task includes:
*   Implementing a 'Forgot Password' screen with an email input to request a reset link.
*   Connecting this UI to the /auth/forgot-password backend API endpoint.
*   Implementing a 'Reset Password' screen where users can enter a new password using a valid token from the email link.
*   Connecting this UI to the /auth/reset-password backend API endpoint.
*   Displaying clear instructions and error messages throughout the process.
